# Manual Testing Sistem LKM Matematika

## KAM

- [ ] Mahasiswa bisa mengerjakan KAM.
- [ ] Setelah submit, skor tampil.
- [ ] Setelah submit, jawaban benar tampil.
- [ ] KAM nomor 1 menampilkan cm².
- [ ] KAM nomor 9 menampilkan 180°.
- [ ] Jika skor di bawah threshold, modul remedial muncul.
- [ ] Tombol Buka Modul berfungsi.
- [ ] Tombol Unduh Modul berfungsi.

## Pre Test

- [ ] Mahasiswa bisa mengerjakan Pre Test.
- [ ] Setelah submit, jawaban tersimpan.
- [ ] Jawaban benar tidak tampil untuk mahasiswa.
- [ ] Pembahasan tidak tampil untuk mahasiswa.
- [ ] Network response mahasiswa tidak berisi correctAnswer.
- [ ] Network response mahasiswa tidak berisi explanation.
- [ ] Dosen/admin tetap bisa melihat kunci.

## Post Test

- [ ] Mahasiswa bisa mengerjakan Post Test.
- [ ] Sebelum submit, kunci tidak tampil.
- [ ] Sebelum submit, pembahasan tidak tampil.
- [ ] Setelah submit, jawaban mahasiswa tampil.
- [ ] Setelah submit, jawaban benar tampil.
- [ ] Setelah submit, status benar/salah tampil.
- [ ] Setelah submit, pembahasan tampil.

## Gambar Pre/Post

- [ ] Nomor 2 tidak broken image.
- [ ] Nomor 3 tidak broken image.
- [ ] Nomor 4 tidak broken image.
- [ ] Nomor 6 tidak broken image.
- [ ] Nomor 10 tidak broken image.

## Format Matematika

- [ ] cm² tampil benar.
- [ ] m² tampil benar.
- [ ] m³ tampil benar.
- [ ] dm³ tampil benar.
- [ ] satuan² tampil benar.
- [ ] 90° tampil benar.
- [ ] 180° tampil benar.
- [ ] 7,38 tampil tanpa spasi salah.

## LKM

- [ ] LKM 1 bisa dibuka.
- [ ] LKM 2 bisa dibuka.
- [ ] LKM 3 bisa dibuka.
- [ ] LKM 4 bisa dibuka.
- [ ] LKM 5 bisa dibuka.
- [ ] LKM 6 bisa dibuka.
- [ ] Mahasiswa bisa mengisi jawaban/refleksi.
- [ ] Mahasiswa bisa submit.
- [ ] Tidak ada skor pada LKM.
- [ ] Tidak ada benar/salah pada LKM.

## Feedback Dosen

- [ ] Dosen bisa membuka halaman Feedback LKM Mahasiswa.
- [ ] Dosen bisa memilih mahasiswa.
- [ ] Dosen bisa melihat LKM 1-6 mahasiswa.
- [ ] Dosen bisa memberi feedback.
- [ ] Dosen bisa mengedit feedback.
- [ ] Mahasiswa bisa melihat feedback dosen.

## Role Security

- [ ] Mahasiswa tidak bisa akses halaman dosen.
- [ ] Mahasiswa tidak bisa akses endpoint admin/guru.
- [ ] Mahasiswa tidak bisa melihat kunci Pre Test.
- [ ] Dosen/admin bisa melihat data yang diperlukan.
