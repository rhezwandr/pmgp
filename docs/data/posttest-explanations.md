# Post Test Explanations - Catatan Implementasi

Sumber utama: `docs/source/Jawban dan Pembahasan untuk di tunjukan saat Post test.docx`

Aturan:

1. Pembahasan hanya tampil setelah mahasiswa submit Post Test.
2. Pembahasan tidak tampil pada Pre Test untuk mahasiswa.
3. Dosen/admin boleh melihat pembahasan jika diperlukan.
4. Format matematika wajib dirapikan:
   - m3 → m³
   - m2 → m²
   - cm2 → cm²
   - dm3 → dm³
   - satuan2 → satuan²
   - 90o/90O → 90°
   - 180o/180O → 180°
   - 7, 38 → 7,38

Ringkasan pembahasan penting:

- Nomor 1: hasil akhir 7,38 m³.
- Nomor 2: gunakan luas segitiga, satuan m².
- Nomor 3: luas papan/bidang, satuan m².
- Nomor 4: volume 6000 dm³ = 6 m³, debit 4 m³/menit, waktu 1,5 menit.
- Nomor 5: volume akhir 12 m³.
- Nomor 6: luas daerah diarsir 25 satuan².
- Nomor 7: rotasi 90° searah jarum jam.
- Nomor 8: ∠DEF = 95°.
- Nomor 9: jawaban sesuai file sumber.
- Nomor 10: luas seluruh bidang 38,5 cm².
